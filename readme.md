\# Документация: Генератор бейджиков для печати на А4



Программа представляет собой полнофункциональное настольное приложение на Python с графическим интерфейсом для автоматизированного создания персонализированных бейджиков и их размещения на листах формата А4 для печати. \[techtrek](https://www.techtrek.io/badge-generator-in-python/)



\## Назначение

Автоматизирует процесс подготовки бейджиков для конференций, семинаров и мероприятий университета. Позволяет загружать шаблон, настраивать текст (ФИО, должность), подгружать списки участников и генерировать готовые к печати страницы с оптимальной версткой.



\## Требования

\- Python 3.8+.

\- Библиотеки: `tkinter` (встроенная), `Pillow` (`pip install Pillow`).

\- Шрифты TTF/OTF (опционально).

\- Изображение-шаблон бейджика (PNG/JPG).



\## Установка и запуск

```python

pip install Pillow

python badge\_generator.py

```

Окно откроется размером 1400x900 пикселей.



\## Интерфейс (вкладки Notebook)

Приложение использует `ttk.Notebook` для трех вкладок. \[stackoverflow](https://stackoverflow.com/questions/44745297/adding-notebook-tabs-in-tkinter-how-do-i-do-it-with-a-class-based-structure)



| Вкладка     | Назначение                          |

|-------------|-------------------------------------|

| \*\*Шаблон\*\* | Настройка изображения и областей текста. |

| \*\*Печать\*\* | Параметры верстки на А4.            |

| \*\*Предпросмотр\*\* | Навигация по сгенерированным страницам. |



\## Основные классы и методы



\### BadgeGenerator (основной класс)

Инициализирует окно, загружает настройки из `badge\_settings.json`, создает UI.



\*\*Ключевые атрибуты:\*\*

\- `template\_image`: PIL.Image шаблона бейджика.

\- `name\_area`, `position\_area`: Кортежи (x1,y1,x2,y2) для областей текста.

\- `students`: Список кортежей `\[(ФИО, должность), ...]`.

\- Параметры печати: `a4\_width=2480px`, `a4\_height=3508px` (300 DPI), `badge\_width/height`, отступы.



\*\*Методы инициализации:\*\*

\- `load\_settings()`: Загружает JSON-настройки (размеры, цвет, шрифт, области).

\- `save\_settings()`: Сохраняет при закрытии.

\- `setup\_ui()`: Создает Notebook с тремя Frame.



\### Вкладка "Шаблон" (setup\_template\_tab)

\- \*\*Загрузка шаблона\*\* (`load\_template`): Открывает изображение, масштабирует на Canvas, сохраняет метаданные (масштаб, координаты).

\- \*\*Выделение областей\*\*: Мышью на Canvas (`start\_selection`, `update\_selection`, `end\_selection`). Учитывает масштаб изображения для точных координат в пикселях шаблона. \[stackoverflow](https://stackoverflow.com/questions/1970807/center-middle-align-text-with-pil)

\- \*\*Настройки текста\*\*:

&nbsp; - Шрифт: Combobox с системными или кастомным (`load\_custom\_font`, `get\_font` с ImageFont.truetype).

&nbsp; - Цвет: colorchooser.

\- \*\*Тест\*\*: Поля для ФИО/должности, кнопка предпросмотра (`show\_preview`).

\- \*\*Загрузка данных\*\*: TXT-файлы ФИО/должностей (`load\_names`, `load\_positions`, `merge\_data`).



\*\*draw\_centered\_text\*\*: Рисует центрированный текст на ImageDraw с bbox для точного позиционирования. \[pillow.readthedocs](https://pillow.readthedocs.io/en/stable/reference/ImageDraw.html)



\### Вкладка "Печать" (setup\_print\_tab)

Spinbox для размеров в мм (преобразование в пиксели: `mm\_to\_pixels(mm, dpi=300)`).

\- \*\*calculate\_layout\*\*: Рассчитывает сетку (столбцы/строки) на А4: `cols = (avail\_w + spacing\_x) // (badge\_w + spacing\_x)`.

\- \*\*generate\_a4\_pages\*\*: 

&nbsp; - Создает Image.new('RGB', A4-размер).

&nbsp; - Для каждого студента: копирует шаблон, рисует текст, ресайзит, пастит на страницу с линиями обреза.

&nbsp; - Прогресс-бар, сохранение PNG (300 DPI), миниатюры для предпросмотра. \[e-badges](https://www.e-badges.net/button-badge-generator-in-python/)



\### Вкладка "Предпросмотр" (setup\_preview\_tab)

Scrollable Canvas с навигацией (◀/▶). `update\_preview` отображает thumbnail страниц.



\## Алгоритм работы

1\. Загрузите шаблон → Выделите области ФИО/должности (или вручную).

2\. Настройте шрифт/цвет → Протестируйте.

3\. Загрузите TXT: ФИО (по строкам), должности.

4\. В "Печать": Установите размеры → Рассчитайте → Создайте листы.

5\. Просмотрите/сохраните PNG в папку.
